games worden gemaakt aan de hand van de Game class

de naam van een Game classe moet beginnen met Game_

de game moet in de default package zijn.

de classes in dit project zijn classes die worden gebruikt door de host aplicatie. als je hier een object van wil maken moet dat gedaan
worden dmv de classloader die is meegegeven in de game class. als je een object wil maken van zelf gemaakte classes moet dat ook dmv de
classloader die is meegegeven in de game class.

om de game te testen export als jar niet runable jar naar de games map in dezelfde folder waar de host aplicatie staat. start dan de host
applicatie en selecteer je game in het game menu
